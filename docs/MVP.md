# ProjectOS MVP

## Goal
Build the first working version of ProjectOS.

## Tech Stack
- Nova for development
- GitHub for code
- Supabase for database and authentication
- Vercel for deployment

## MVP Modules
1. Login
2. Dashboard
3. Projects
4. Documents
5. AI Assistant

## Login
Users can sign in with email.

## Dashboard
Dashboard shows:
- Active Projects
- Delayed Projects
- Open Risks
- Pending Documents
- AI Summary
- Smart Notifications

## Projects
Users can:
- View project list
- Create new project
- Open project detail page
- See project status, progress, budget and risks

## Documents
Users can:
- Upload documents
- View document list
- Search documents

## AI Assistant
Users can ask:
- Show delayed projects
- Prepare weekly report
- Summarize project status
- List urgent actions

## Design
- Premium dark SaaS interface
- Purple and indigo accent colors
- Left sidebar navigation
- Clean enterprise dashboard
- Responsive layout

## First Release
The first release should focus on usability, clean UI and core workflow only.
